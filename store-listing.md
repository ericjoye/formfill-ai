# FormFill AI — Chrome Web Store Listing

## Extension Name

**FormFill AI — Smart Auto-Fill for Web Forms**

## Short Description (132 chars max)

AI-powered form filler that auto-fills web forms from your saved profile. Works on React, Vue, and complex JS forms.

## Long Description

FormFill AI is a Chrome extension that uses AI to intelligently auto-fill web forms from a saved user profile. Unlike Chrome's built-in autofill, FormFill AI understands form context — it matches fields using labels, placeholders, ARIA attributes, autocomplete values, and input types. It works on modern React, Vue, and JavaScript-heavy forms where standard autofill fails.

**SMART FIELD DETECTION**
FormFill AI doesn't just match field names — it uses multi-strategy detection to understand form context. It handles text inputs, email, phone, dropdowns, textareas, and more. React-compatible value injection ensures fields are actually filled, not just visually changed. The result: accurate form filling on the sites that matter most.

**SAVE YOUR PROFILE ONCE**
Set up your profile with name, email, phone, address, LinkedIn, and other fields. FormFill AI stores everything locally in your browser using `chrome.storage.local` — your data never leaves your device. No accounts, no sign-ups, no cloud storage.

**PRIVACY-FIRST WITH AI**
FormFill AI processes form data locally in your browser. When AI processing is needed, API calls go directly from your browser to your chosen AI provider. You can bring your own API key for full control. No form data is sent to our servers at any time.

**TIERED PRICING**
- **Free tier:** Basic form filling with standard field detection
- **Pro tier:** AI-powered context-aware filling, priority support, and advanced features

## Key Features

- **AI-powered field detection** — multi-strategy matching using labels, placeholders, ARIA, and autocomplete
- **React/Vue compatible** — works on modern JavaScript frameworks where standard autofill fails
- **Local profile storage** — your data stays in your browser, never sent to external servers
- **BYOK privacy model** — bring your own API key for AI processing
- **No account required** — works out of the box with local storage
- **Manifest V3** — fully compatible with the latest Chrome extension standard

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select the `formfill-ai/` folder
5. The FormFill AI icon appears in your toolbar

## Requirements

- Chrome 88+ (Manifest V3 support)
- Optional: API key for AI-powered features (bring your own)
